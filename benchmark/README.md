# benchmark/

Holds the **participant-facing** benchmark.

- `public/` is generated by `scripts/prepare_public_benchmark.py` and is the
  only thing you hand to participants. It contains:
  - `cases/` — the selected `BenchmarkTestNNNNN.java` files to audit.
  - `report_template.json` — the JSON schema to fill in (one per case).
  - `README.md` — participant-facing instructions (auto-generated).

Nothing under `public/` reveals a ground-truth label. The answer key
(`expectedresults-1.2.csv`) and the selection metadata live under `organizer/`
and must never be copied here.

Regenerate after changing the selection:

```bash
python scripts/select_cases.py
python scripts/prepare_public_benchmark.py
```
